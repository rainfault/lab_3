Проект A-Frame + GLB

Файлы:
- index.html — основной файл сцены.
- scene.glb — сюда помести свой файл сцены (переименуй его в scene.glb).

Как использовать:
1. Положи свой .glb файл в эту же папку и назови его scene.glb.
2. Открой index.html в браузере (Chrome/Firefox).
3. В сцене можно вращать камеру мышью и ходить WASD.

Если хочешь указать другое имя файла:
- открой index.html
- найди строку: src="scene.glb"
- замени на нужный путь (например, models/my_scene.glb).
